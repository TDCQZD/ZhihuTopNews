# ZhihuTopNews
#高仿知乎日报首页

* 架构：MVP

* 图片加载：Glide

* 网络请求：RxJava & Retrofit＋okhttp

* 其他：Cardview，RecycleView，Butterknife


#完成功能

* 文章列表的展示与缓存


* 首页界面的轮播控件

* 文章浏览到底部时自动加载更多

* 标记已读文章

* 自动缓存文章

* 夜间模式

后续还会完善更新。。。。
